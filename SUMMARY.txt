🚀 快速开始（仅需 5 步）
════════════════════════════════════════════════════════════════════════════

【第 1 步】推送到 GitHub (3 分钟)
────────────────────────────────────────────────────────────────────────────
  git init
  git add .
  git commit -m "Initial: ZhiQin Attendance System"
  git branch -M main
  git remote add origin https://github.com/YOUR_USERNAME/zhiqin-attendance.git
  git push -u origin main

【第 2 步】部署到 Vercel (2 分钟)
────────────────────────────────────────────────────────────────────────────
  1. 访问 https://vercel.com
  2. 用 GitHub 登录
  3. "New Project" → 选择仓库 → "Deploy"
  ✅ 自动部署，获得链接：https://zhiqin-attendance.vercel.app

【第 3 步】生成 Android APK (10 分钟)
────────────────────────────────────────────────────────────────────────────
  chmod +x build-release-apk.sh
  ./build-release-apk.sh
  ✅ 输出：android/app/release/app-release.apk

【第 4 步】上传 APK 到 Vercel (2 分钟)
────────────────────────────────────────────────────────────────────────────
  cp android/app/release/app-release.apk public/app.apk
  git add public/app.apk
  git commit -m "build: Add Android APK"
  git push origin main
  ✅ 自动部署：https://zhiqin-attendance.vercel.app/app.apk

【第 5 步】分享给团队 (0 分钟)
────────────────────────────────────────────────────────────────────────────
  Web 版：https://zhiqin-attendance.vercel.app
  下载中心：https://zhiqin-attendance.vercel.app/download.html
  APK：https://zhiqin-attendance.vercel.app/app.apk

📋 使用流程
════════════════════════════════════════════════════════════════════════════

【员工首次登录】
  1. 打开 Web 应用链接
  2. 输入邀请码（TEAM2024）和真实姓名
  3. 提交加入申请
  4. 等待管理员审核和激活
  5. 登录后可开始打卡

【管理员审核激活】
  1. 登录应用（自动为 admin）
  2. 点击"全员名册"
  3. 找到待激活用户
  4. 点击"激活"按钮
  5. 员工即可使用

【修改打卡地址】
  1. 管理员登录
  2. 点击"合规设定"标签页
  3. 输入新的纬度、经度、范围
  4. 点击"保存设置"
  5. 所有员工实时使用新配置

【员工打卡】
  1. 打开应用
  2. 允许位置权限
  3. 在允许范围内，点击"上班打卡"
  4. 工作中显示实时计时器
  5. 下班前填写工作汇报
  6. 点击"下班签退"
  7. 完成！
